# イベント一覧ページ用動画フォルダ

このフォルダには、イベント一覧ページのヒーローセクションで使用する動画ファイルを配置してください。

## 対応ファイル形式

- `event-hiro.webm` - WebM形式（推奨）

## 使用方法

1. 動画ファイルをこのフォルダに配置してください
2. ファイル名は `event-hiro.webm` にしてください
3. ブラウザでページをリロードすると、動画が表示されます

## 注意事項

- 動画ファイルが読み込めない場合、ポスター画像が表示されます
- 動画ファイルのサイズが大きい場合は、読み込みに時間がかかる場合があります
